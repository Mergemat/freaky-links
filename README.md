## Development

```bash
bun install
npx convex dev    # terminal 1
bun run dev       # terminal 2
```

## Environment

```env
VITE_CONVEX_URL=https://your-deployment.convex.cloud
```

## Deploy

1. Connect to Vercel
2. Add `VITE_CONVEX_URL` env var
3. Add domains: `freakylinks.sh` + `*.fr34ky.link`
